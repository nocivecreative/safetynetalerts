## Mission
Le but de l’application à développer, SafetyNet Alerts, est d'envoyer des informations aux systèmes de services de secours. Dans ce projet, vous devez développer le back-end de cette application, c'est-à-dire une API REST qui renverra différentes informations en fonction de la demande.
